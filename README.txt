========================
Arduino Relay Controller
========================

Arduino-based relay controller to control 24V solenoids, etc.

firmware
========

Contains the Arduino firmware files. See firmware/README.txt

host
====

Contains the code for controlling with a host computer.

python
------

See host/python/README.txt

pcb
===

Contains the arduino_relay_controller pcb Kicad and gerber files.
